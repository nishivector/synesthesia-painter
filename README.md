# Synesthesia Painter

An audio-visual puzzle game where you paint with colors that emit musical tones. Create harmonious chords to solve puzzles!

## How to Play

- **Keys 1-7** — Select color (Cyan, Magenta, Yellow, Red, Green, Blue, Purple)
- **Click/Drag** — Paint shapes on the canvas
- **Shift+Click** — Erase shapes
- **Scroll** — Zoom in/out

Match the target chord shown at the top to complete each level!

## Features

- 7 colors mapped to musical notes (C4 through A5)
- Bauhaus + liquid chrome aesthetic
- Iridescent metallic materials
- Tone.js synthesized audio
- 10 progressive levels
